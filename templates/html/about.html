About Recipe Maker

The Project

Recipe Maker is a personal project created by Wes Ordonez to make meal planning easier and more efficient. The goal is to help people create delicious meals based on what's available in their local stores' weekly ads.

How It Works

Using n8n automation, Recipe Maker analyzes weekly ads from local stores and generates recipe ideas based on the available ingredients. This helps you:

  • Save money by using ingredients that are on sale
  • Discover new recipes based on available ingredients
  • Plan your meals more efficiently
  • Reduce food waste by using what's available

Roadmap

Phase 1: Basic Recipe Generation

  • Generate recipe ideas from weekly ads
  • Display recipes in a clean, readable format
  • Basic dark/light mode support

Phase 2: Enhanced Features

  • User accounts and preferences
  • Save favorite recipes
  • Customize recipe generation based on dietary restrictions
  • Add more store weekly ads support

Phase 3: Advanced Features

  • Meal planning calendar
  • Shopping list generation
  • Nutritional information
  • Recipe scaling for different serving sizes
